# Library Management System
The term project CS-534 Advanced-OOP.

### Prerequisite
MySQL 8.0.13 

### Installing
MySQL can be installed quickly using HomeBrew.

```
brew update
brew install mysql
```

After installation: 
```
brew services start mysql
```

Optional: MySQLWorkbench
